# 🐍 Neon Snake Game

A visually stunning, modern take on the classic Snake game! Built completely with Vanilla JavaScript, HTML5 Canvas, and CSS, featuring a beautiful glassmorphism UI, neon glow effects, and newly implemented leveling mechanics.

## ✨ Features

- **Premium Aesthetics**: A deep purple/blue dark mode background layered with vibrant neon pink and cyan game components. The UI utilizes modern CSS properties like `backdrop-filter` for a sleek glassmorphism effect.
- **Leveling System**: Eat 5 apples to progress! But be careful—as your level increases, so does the speed of the game.
- **Dynamic Obstacles**: At higher levels, static neon blockers will spawn on the grid to add an extra layer of difficulty.
- **Wall Wrapping**: No cheap deaths on the edge! Moving off the screen seamlessly wraps the snake to the other side.
- **Speed Control**: Too fast or too slow? Use the slider on the start screen to set your preferred starting difficulty.
- **High Score Tracking**: Rest easy knowing your best runs are locally saved in your browser.

## 🚀 How to Run

Because this game relies purely on frontend web technologies, there's no complex build process or developer environment needed!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/roopashenoymallaya/snake-game.git
   ```

2. **Open the game:**
   Navigate into the project folder and simply open the `index.html` file in your favorite web browser (Chrome, Safari, Firefox, Edge).
   - On Mac: Just double-click the file in Finder, or run `open index.html` in your terminal.

## 🛠️ Technology Stack

- **HTML5 Canvas:** For rendering the high-performance 60fps game loop and applying shadow blur glows.
- **Vanilla JavaScript:** All game logic, grid tracking, collision detection, and level transitions.
- **CSS3:** For interactive overlays, responsive flexing, gradient styling, and custom animations.
- **Google Fonts:** Utilizing the beautifully geometric **[Outfit](https://fonts.google.com/specimen/Outfit)** font.

Enjoy your retro-neon experience! Let's see how high you can climb the levels.
